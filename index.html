📚 Ključne CSS Tehnike

@keyframes

Definira korake animacije

@keyframes ime {
from { opacity: 0; }
to { opacity: 1; }
}

animation

Primjenjuje animaciju

animation: ime 2s ease infinite;

transition

Glatki prijelazi

transition: all 0.3s ease;

transform

Rotacija, skaliranje, pomak

transform: rotate(45deg) scale(1.2);
