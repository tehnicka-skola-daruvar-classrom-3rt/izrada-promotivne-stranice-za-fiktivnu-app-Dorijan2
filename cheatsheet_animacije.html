📖 CSS Animacije Cheatsheet

Sve što trebaš znati o CSS animacijama na jednom mjestu

Transitions @keyframes Animation Transform Timing Primjeri Savjeti

⏩ Transition (Prijelazi)

Transition stvara glatki prijelaz između dva stanja elementa (npr. na hover).

Osnovna sintaksa

.element { transition: property duration timing-function delay; } /* Primjer */ .button { background: blue; transition: background 0.3s ease; } .button:hover { background: red; }

Transition svojstva

Svojstvo Opis Primjer
transition-property Koje CSS svojstvo animirati all, background, transform
transition-duration Trajanje animacije 0.3s, 500ms, 2s
transition-timing-function Brzina kroz vrijeme ease, linear, ease-in-out
transition-delay Odgoda prije početka 0s, 0.5s, 1s

Skraćeni zapis (shorthand)

/* transition: property duration timing-function delay; */ transition: all 0.3s ease 0s; transition: transform 0.5s ease-in-out; transition: background 0.3s, transform 0.5s; /* više svojstava */

🎬 @keyframes (Definiranje animacije)

@keyframes definira korake animacije - od početka do kraja.

Osnovna sintaksa

/* Jednostavno: from/to */ @keyframes imeAnimacije { from { opacity: 0; } to { opacity: 1; } } /* Detaljno: postoci */ @keyframes bounce { 0% { transform: translateY(0); } 50% { transform: translateY(-30px); } 100% { transform: translateY(0); } }

Napredni primjeri

/* Više svojstava odjednom */ @keyframes slideIn { from { transform: translateX(-100%); opacity: 0; } to { transform: translateX(0); opacity: 1; } } /* Više koraka */ @keyframes colorChange { 0% { background: red; } 25% { background: yellow; } 50% { background: green; } 75% { background: blue; } 100% { background: red; } }

▶ Animation (Primjena animacije)

Animation svojstvo primjenjuje @keyframes animaciju na element.

Sva animation svojstva

Svojstvo Opis Vrijednosti
animation-name Ime @keyframes animacije bounce, slideIn, none
animation-duration Trajanje jednog ciklusa 1s, 500ms, 2.5s
animation-timing-function Krivulja brzine ease, linear, steps(5)
animation-delay Odgoda prije početka 0s, 1s, -0.5s
animation-iteration-count Broj ponavljanja 1, 3, infinite
animation-direction Smjer animacije normal, reverse, alternate
animation-fill-mode Stanje prije/poslije none, forwards, backwards, both
animation-play-state Pauza/play running, paused

Skraćeni zapis (shorthand)

/* animation: name duration timing-function delay iteration-count direction fill-mode; */ animation: bounce 1s ease infinite; animation: slideIn 0.5s ease-out forwards; animation: pulse 2s ease-in-out 0.5s infinite alternate; /* Više animacija odjednom */ animation: bounce 1s infinite, fadeIn 0.5s forwards;

animation-direction vrijednosti

normal

0% → 100%

Uvijek naprijed
reverse

100% → 0%

Uvijek unatrag
alternate

0% → 100% → 0%

Naprijed-nazad
alternate-reverse

100% → 0% → 100%

Nazad-naprijed

animation-fill-mode vrijednosti

none Vraća se na početno stanje
forwards Ostaje na završnom stanju
backwards Primjeni početno stanje za vrijeme delaya
both forwards + backwards

🔄 Transform (Transformacije)

Transform mijenja oblik, poziciju i veličinu elementa. Često se koristi s animacijama.

Transform funkcije

Funkcija Opis Primjer
translateX() Pomak horizontalno translateX(50px), translateX(-100%)
translateY() Pomak vertikalno translateY(-20px), translateY(50%)
translate() Pomak X i Y translate(50px, 20px)
rotate() Rotacija rotate(45deg), rotate(-90deg)
scale() Skaliranje scale(1.5), scale(0.5), scale(2, 1)
scaleX() / scaleY() Skaliranje po osi scaleX(2), scaleY(0.5)
skewX() / skewY() Nagib/iskrivljenje skewX(20deg), skewY(-10deg)
rotateX() / rotateY() 3D rotacija rotateX(180deg), rotateY(45deg)

Vizualni pregled

rotate(45deg)
scale(0.7)
skewX(20deg)
translateX(10px)

Kombiniranje transformacija

/* Više transformacija u jednoj liniji */ transform: translateX(50px) rotate(45deg) scale(1.2); /* Redoslijed je bitan! */ transform: rotate(45deg) translateX(50px); /* različit rezultat */ transform: translateX(50px) rotate(45deg); /* od ovog */

Transform origin

/* Točka oko koje se transformira (default: center) */ transform-origin: center; /* default */ transform-origin: top left; /* gornji lijevi kut */ transform-origin: 50% 100%; /* donji centar */ transform-origin: 0 0; /* gornji lijevi kut */

⏱ Timing Functions (Krivulje brzine)

Određuju kako se animacija ubrzava/usporava kroz vrijeme.

Ugrađene funkcije

Vrijednost Opis Kada koristiti
ease Sporo → brzo → sporo Default, prirodno izgleda
ease-in Sporo → brzo Elementi koji izlaze
ease-out Brzo → sporo Elementi koji ulaze
ease-in-out Sporo → brzo → sporo Kontinuirane animacije
linear Konstantna brzina Rotacije, loaderi
steps(n) n diskretnih koraka Sprite animacije, typing

Usporedba timing funkcija uživo

ease
ease-in
ease-out
ease-in-out
linear

Custom cubic-bezier

/* cubic-bezier(x1, y1, x2, y2) */ transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); /* bounce */ transition-timing-function: cubic-bezier(0.175, 0.885, 0.32, 1.275); /* elastic */ /* Alat za kreiranje: cubic-bezier.com */

Steps() funkcija

/* steps(broj_koraka, jump-term) */ animation-timing-function: steps(5); /* 5 koraka */ animation-timing-function: steps(10, end); /* skok na kraju koraka */ animation-timing-function: steps(1, start); /* instant promjena */ /* Primjer: typing efekt */ @keyframes typing { from { width: 0; } to { width: 100%; } } .text { animation: typing 3s steps(20) forwards; }

💡 Gotovi Primjeri (Copy-Paste)

Fade In

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } .element { animation: fadeIn 0.5s ease forwards; }

Slide In From Left

@keyframes slideInLeft { from { transform: translateX(-100%); opacity: 0; } to { transform: translateX(0); opacity: 1; } } .element { animation: slideInLeft 0.5s ease-out forwards; }

Bounce

@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } .element { animation: bounce 1s ease infinite; }

Pulse

@keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.1); } } .element { animation: pulse 2s ease-in-out infinite; }

Shake

@keyframes shake { 0%, 100% { transform: translateX(0); } 10%, 30%, 50%, 70%, 90% { transform: translateX(-5px); } 20%, 40%, 60%, 80% { transform: translateX(5px); } } .element { animation: shake 0.5s ease; }

Spin (Rotacija)

@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .loader { animation: spin 1s linear infinite; }

Hover Lift Effect

.card { transition: transform 0.3s ease, box-shadow 0.3s ease; } .card:hover { transform: translateY(-10px); box-shadow: 0 20px 40px rgba(0,0,0,0.2); }

Gradient Animation

@keyframes gradient { 0% { background-position: 0% 50%; } 50% { background-position: 100% 50%; } 100% { background-position: 0% 50%; } } .element { background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab); background-size: 400% 400%; animation: gradient 15s ease infinite; }

📚 Savjeti i Best Practices

✅ Koristi transform i opacity
Ova svojstva su hardverski akcelerirana i najperformantnija za animacije.
✅ Koristi will-change za kompleksne animacije
will-change: transform; priprema browser za animaciju.
❌ Izbjegavaj animirati width, height, top, left
Ova svojstva uzrokuju layout reflow i usporavaju stranicu. Koristi transform umjesto toga.
❌ Ne pretjeruj s animacijama
Previše animacija odvlači pažnju i može uzrokovati mučninu kod korisnika.
✅ Poštuj prefers-reduced-motion
Neki korisnici žele smanjene animacije:
@media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } }

Preporučena trajanja

Tip animacije Preporučeno trajanje
Hover efekti 0.2s - 0.3s
Modali, dropdowni 0.3s - 0.4s
Page transitions 0.4s - 0.6s
Loading animacije 1s - 2s (beskonačne)
Attention grabbers 0.5s - 1s

Svojstva koja se mogu animirati

/* DOBRO - hardverski akcelelirano */ transform, opacity /* OK - ali sporije */ color, background-color, border-color box-shadow, text-shadow filter, backdrop-filter /* IZBJEGAVATI - uzrokuje reflow */ width, height, top, left, right, bottom margin, padding, font-size
